# C_Learning

Ce dossier contient mes exercices d'apprentissage du 
langage C. Chaque dossier correspond généralement 
à un concept (variables, conditions, fonctions, 
mini‑projets, etc.).

---

## Structure
- `01_Hello_World` : premiers pas
- `02_Variables` : variables, types et spécificateurs 
de format
- `03_Input` : entrée utilisateur
- `04_Math_Functions` : fonctions mathématiques 
basiques
- `05_Conditions_Switch` : conditions if/else/elif + 
switch (un fichier dédié contient principalement 
des commentaires explicatifs)
- Le reste viendra plus tard

---

## Compilation
Chaque dossier contient un fichier `build.sh` 
pour compiler le fichier `main.c`.  
À exécuter dans le terminal :

```bash
./build.sh
```
Puis exécuter l'exécutable `program` :

```bash
./program
```
